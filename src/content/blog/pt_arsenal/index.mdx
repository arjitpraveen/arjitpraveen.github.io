<Callout variant="warning">

Running `pipx install git+https://github.com/Pennyw0rth/NetExec` might throw an error if your python version is incompatible with some dependencies. In that case, you'll need to use `pyenv` to switch between python versions. You must also suffix `pipx install git+https://github.com/Pennyw0rth/NetExec ` with `--python "$(pyenv which python)"` 

```bash
sudo dnf install -y libffi-devel sqlite-devel readline-devel ncurses-devel bzip2-devel xz-devel tk-devel
pyenv install 3.13.2
pyenv global 3.13.2
```

</Callout>
